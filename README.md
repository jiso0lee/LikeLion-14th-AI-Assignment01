# LikeLion-14th-Assignment-Template
🦁 SKHU 멋쟁이사자처럼 14기 과제 PR 템플릿 레포지토리입니다.
1. 선택한 서비스: 실시간 교통 예측 서비스
2. 서비스의 예측: 사용자가 이동할 때 걸리는 시간(예상 도착 시간)과 최적의 경로를 예측합니다.
3. 사용되는 data: GPS 데이터 - 도로나 철도 위 차량 속도 파악, 과거 교통 패턴 데이터 - 시간대별 정체 정도 학습 (특히 출퇴근 시간대 혼잡한 교통 상황 학습), 사용자별 이동 경로 - 실제 사람들이 선택한 이동 경로 반영, 날씨나 사고 정보 - 변수로 인한 교통 흐름 변화 파악
4. 모델의 학습 방법: 이 모델은 입력받은 GPS, 과거 교통 패턴 , 시간 데이터 등을 입력 받아서 실제 이동 시간을 추측하는 방법으로 운영되기 때문에 지도 학습입니다.
5. 모델의 개선점: 실시간 데이터 확보 - 현재 교통 상황을 빠르게 파악하면 정확도가 증가합니다.
